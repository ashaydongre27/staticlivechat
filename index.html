🎲 Random Number to GitHub CSV

Store random numbers directly in your GitHub repository
📝 Setup Instructions:
  1. Create a Personal Access Token: GitHub Settings → Developer Settings → Personal Access Tokens → Generate New Token (Classic)
  2. Select scope: repo (Full control of private repositories)
  3. Copy the token and paste it below
  4. Enter your GitHub username and repository name
  5. The CSV file will be created automatically at data.csv

⚙️ GitHub Configuration

Keep this secret! It's stored only in your browser.
